<h1 align="center">Assets</h1>

This repository contains all assets regarding branding and theming of project [Docuisine](https://github.com/docuisine).

## Directory Structure

```bash
.
├── branding                    # All images used for branding.
│   ├── favicon                 # Circular logos for the web app.
│   ├── logo                    # Main logo and design.
│   └── typography              # Word version for the logo for Docuisine.
└── src                         # Source / project files of the images (.psd).
```
